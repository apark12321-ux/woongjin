# 웅진 수식 조판 규칙 (전문)

> 208개 샘플 실측으로 확정. 한글을 제외한 전부는 수식편집기(Ctrl+N,M / 10pt).
> **수식 안에 한글을 넣지 않는다.** 영문 점 라벨/변수/도형명도 전부 수식편집기 처리.

`│` 는 개체(수식 객체) 분할 경계를 뜻하는 표기다.

---

## ① 관계연산자 `=` 에서 분할 (HEAD 모드)

`=`에서 분할하되, `=`는 **우변 개체의 머리에 붙인다.** `=`가 여러 번이면 매번 분할한다.

- `AB │ =AC`
- `∠B │ =70°`
- `∠x │ =90°−36°−36° │ =18°`  ← `=`마다 분할

## ② 산술·비는 통째 유지

산술(±, ×, ÷, 분수)과 비(`:`)는 쪼개지 않고 한 개체 안에 통째로 둔다.

- `x+y` → 통째
- `∠x:∠y:∠z │ =3:7:5`  ← `=`에서만 분할, `3:7:5`는 통째

## ③ ⊥(PERP) · ≡(EQUIV) 는 좌변과 병합

`PERP`·`EQUIV`는 **좌변 피연산자와 한 개체로 병합**한다.
조각내면 기호로 렌더되지 않으므로 반드시 한 객체로 묶는다.

- `l PERP n`
- `bar{rmAB} PERP bar{rmDE}`
- `TRIANGLE rmABC EQUIV TRIANGLE rmDCB`

## ④ 표기 규칙

| 대상 | 표기 |
|---|---|
| 각·삼각형·수직·합동·곱·도 | `ANGLE` `TRIANGLE` `PERP` `EQUIV` `TIMES` `DEG` (전부 대문자) |
| 선분 | `bar{...}` |
| 반직선 | `vec{...}` |
| 직선 | `dyad{...}` |
| 분수 | `{a}over{b}` |
| 점이름·단위·합동·면 | `rm` (정체) |
| 소문자 변수·직선이름(l·m·n) | 이탤릭 (rm 붙이지 않음) |

## ⑤ 호(arc)

- 기호 호: `rm arch XX` (정체)
- 명사 '호': 텍스트로 유지 (수식화하지 않음)

## ⑥ 숫자–단위 간격

숫자와 단위 사이는 **한 칸** 띄운다.

- `9 rm cm`  (O)
- `9rmcm`   (X)

## ⑦ 미세 간격

- 콤마 뒤: `~` (한 칸)
- 반 칸: `` ` `` (백틱)

## 검증 기준 (납품 전)

- `=` 미분할 0건
- ⊥·≡ 댕글링(좌변과 분리) 0건
- 수식 안 한글 0건
- 숫자-단위 붙여쓰기(`9rmcm` 형) 0건
- 전체 수식 전수검사 통과
